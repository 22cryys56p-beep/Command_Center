# Portfolio Overview

## Portfolio rules

- Each active project has one operational [[Active Projects/Teacher Toolbox/Dashboard|dashboard]], one current milestone, and a visible next action.
- Project detail belongs inside its workspace; portfolio awareness belongs on the Executive Dashboard.
- Move inactive work to `Future Projects/`, and finished work to `Completed Projects/`.

## Status key

| Indicator | Meaning |
| --- | --- |
| 🟢 | On track |
| 🟡 | Needs attention / planning |
| 🔴 | Blocked |
| 🔵 | Paused |
| ⚪ | Complete / archived |

# Teacher Toolbox — Vision

## Purpose

Provide practical tools that help educators spend less time on repetitive administration and more time supporting students.

## First questions to answer

- Who is the primary user for the first release?
- Which single recurring task is most valuable to improve first?
- What outcome makes the first release successful?

## Product principles

- Start with a clear teacher workflow, not a feature list.
- Favor simple, reliable tools over broad but unclear scope.
- Keep student data handling deliberate and privacy-conscious.

## Non-goals for M1

- [ ] Define after the first user workflow is selected.

# Teacher Toolbox Reference

Use this area for supporting detail, not daily operational status.

- `Architecture/` — technical designs and diagrams
- `Design Notes/` — UX and product decisions
- `Technical Reference/` — implementation research
- `User Reference/` — user-facing material
- `Decisions/` — project decision links
- `Meeting Notes/` — meeting records
- `Research/` — investigation notes
- `Work Items/` — notes linked from Kanban cards
